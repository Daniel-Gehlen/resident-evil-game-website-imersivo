# Resident Evil Game - Site Imersivo

Este é um site dedicado à franquia Resident Evil Game por fã, desenvolvido com design imersivo, conteúdo longo e elementos visuais entremeados, seguindo rigorosamente as especificações do wireframe fornecido.

## Estrutura de Arquivos

```markdown
resident-evil-site/
├── index.html # Página principal
├── assets/
│ ├── css/
│ │ ├── style.css # Arquivo principal que importa todos os módulos
│ │ ├── base/ # Estilos base
│ │ │ ├── variables.css # Variáveis CSS (cores, fontes, etc)
│ │ │ ├── reset.css # Reset e estilos globais
│ │ │ └── typography.css # Estilos tipográficos
│ │ └── components/ # Componentes modulares
│ │ ├── header.css # Cabeçalho e subtítulo
│ │ ├── nav.css # Navegação
│ │ ├── featured-image.css # Imagem de destaque
│ │ ├── quote.css # Bloco de citação
│ │ ├── graphic.css # Gráfico/infografia
│ │ ├── video.css # Wrapper de vídeos
│ │ ├── comments.css # Seção de comentários
│ │ └── footer.css # Rodapé
│ ├── js/
│ │ └── main.js # JavaScript (verificação de vídeos e formulário)
│ └── images/ # (opcional) imagens locais
└── README.md # Este arquivo
```

### Características Implementadas

- ✅ **Design idêntico à prova de conceito** - mesma aparência, comportamento e elementos
- ✅ **Arquitetura modular limpa** - componentes separados, sem complexidade
- ✅ **Nomenclatura consistente** - mesmos nomes de classes em toda a estrutura
- ✅ **Paleta de cores Resident Evil** - vermelho acento (#b43b3b), fundos escuros, dourado (#c9a45b)
- ✅ **Acessibilidade visual** - contraste adequado (WCAG AA), foco visível
- ✅ **Vídeos oficiais embedados** - todos de canais oficiais (Capcom/PlayStation)
- ✅ **Elementos do wireframe**:
  - Título e subtítulo
  - Imagem de abertura
  - Citação em destaque visual
  - Gráfico/infografia (SVG)
  - Dois vídeos relacionados
  - Seção de comentários funcional

### Como Usar

1. Clone ou baixe todos os arquivos mantendo a estrutura de diretórios
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Navegue pelo conteúdo imersivo sobre a franquia Resident Evil

### Funcionalidades JavaScript

- Verificação automática de carregamento dos vídeos embedados
- Formulário de comentários funcional (adiciona novos comentários à lista)
- Prevenção de reload no envio do formulário (demo)

### Conteúdo Abordado

O site cobre exaustivamente:

1. **A Gênese do Mal (Pré-1960 - 1998)** - Progenitor, Umbrella, Projeto Wesker, T-Vírus, Incidente na Mansão (RE0 e RE1)
2. **A Queda de Raccoon City (1998)** - RE2, RE3, Outbreak, Code: Veronica
3. **A Era Global do Bioterrorismo (2004-2009)** - RE4, Revelations, RE5
4. **Novas Ameaças e Conexões (2011-2013)** - Revelations 2, RE6
5. **A Saga dos Winters (2017-2037)** - RE7, Village, Shadows of Rose
6. **O Retorno às Origens (2026)** - Resident Evil Requiem
7. **Guia de Plataformas e Jogabilidade no Linux** - Steam Play, Proton, ProtonDB, multiplayer

### Notas de Desenvolvimento

- Alguns vídeos incorporados são de canais de fãs/entusiastas, utilizados na ausência de trailers oficiais disponíveis para incorporação. Todos os links foram testados e estão funcionais no momento da publicação.
- O design é totalmente responsivo e fluido
- A modularização permite fácil manutenção e escalabilidade
- As cores e contrastes foram testados para garantir legibilidade
- **O conteúdo agora reflete fielmente toda a cronologia, personagens e relações da franquia**, conforme a análise detalhada fornecida
